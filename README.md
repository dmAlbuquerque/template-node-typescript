# Template Node Typescript

## Clonando o Projeto
```shell
git clone https://github.com/dmAlbuquerque/template-node-typescript.git
```

<br>

## Instalando as Dependências
```shell
npm install
```

<br>

## Converter Typescript em Javascript
```shell
npm run build
```
